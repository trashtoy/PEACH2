このオブジェクトが取り扱う Timestamp オブジェクトの時差を分単位でセットします. このメソッドは expires 属性の出力に影響します. PHP の date.timezone 設定がシステムの時差と異なる場合に使用してください. 通常はこのメソッドを使用する必要はありません.
